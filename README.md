# CodeGig

> Simple Job find app for coders. This app uses Node, Express, Sequalize(SQLite) and Handlebars.

<kbd>
  <img src="https://i.imgur.com/ebKoG8p.gif"/>
</kbd>


### Version
1.0.0

## Usage

Install Dependencies

```sh
$ npm install
```

Run Server

```sh
$ npm start
```

Visit http://localhost:5000
